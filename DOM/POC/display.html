Block Elements

The elements which occupy the complete horizontal space and does not let any other element occupy their horizontal space. We can specfy height and width of these elements. ex: div, p

Inline Elements

These elements let other elements occupy their horizontal space and does not let us to specify height and width. ex: span

Inline-block Elements
